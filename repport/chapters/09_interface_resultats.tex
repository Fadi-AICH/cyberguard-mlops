\chapter{Interface SOC et résultats}

\section{Interface Streamlit}
L'interface \tooltag{Streamlit} transforme les artefacts ML en outil utilisable par un analyste \cite{streamlit_app}. Elle contient une vue SOC, une file d'événements, un module de live scoring, un onglet preuves modèle et un export de note d'incident.

L'interface a été ajoutée pour montrer la couche utilisateur finale. Un modèle MLOps ne doit pas rester invisible dans un terminal : il doit être exploitable par un profil métier. La file analyste permet de filtrer les événements critiques, le live scoring interroge l'API FastAPI et l'onglet Model Evidence rassemble les métriques, la validation et le drift.
